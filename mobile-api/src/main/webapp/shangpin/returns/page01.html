退货时限 7 天

1、退换货时限是从商品签收之日起开始计算，尚品网会根据您的物流单号帮您确认您的签收时间。

2、箱包皮具类商品自签收之日起7天内，在保证商品本身性能完好，品相全新，未经使用，相关附属配件(如包装、吊牌、卡证等)齐全，不影响我公司二次销售的前提下，提供商品购买凭证，便可以申请退货。

3、自签收之日起7天内，如出现国家三包所规定的质量问题时，请您在第一时间与尚品网联系，经尚品网检验质量问题属实后，我们会为您维修或者办理退货。

4、如超出七天的时间，可以在保修期享受有偿维修服务。

特别提示：对于带有安全封条的箱包皮具类商品，如安全封条破损则无法为您办理退货，请您务必保证安全封条的完整性。
