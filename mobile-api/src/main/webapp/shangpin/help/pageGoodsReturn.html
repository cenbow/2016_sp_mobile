一、如何办理退换货

如出现非人为质量问题，您可向尚品网提出退换货申请，核实为非人为质量问题，会为您办理退换货，但由于尚品网为限量折扣售卖，在无货为您更换的情况下，会及时为您办理退货。
在收到您寄回来的商品后需要2—3个工作日质检，质检后符合尚品网维修退换货政策后会与您联系办理退换货事宜。

二、维修退换货政策

尚品网为限时折扣网站，只提供非人为质量问题的维修退换货服务，由于尚品限时限量售卖，如无可为您更换的产品，将会为您维修或办理退款。同时也请您在商品送达的时候，在配送人员还在场的情况下，当面验收您的商品。在验收过程中发生任何问题请您及时与我们联系，我们会第一时间为您处理。

三、退款说明

尚品网收到您的退货商品后，会有2-3个工作日的质检时间，并在确定质检结果之后，第一时间跟您联系确认。如果符合网站维修退换货政策，款项会在3-15个工作日到达您的账户内，根据不同平台，到账的时间也会有所差异。
温馨提示：在您的订单状态为交易全部完成之后，尚品网方可完成您的退款，因此如果您需要退款请点击确认收货，货到付款形式需等待系统变更状态才可退款，如有疑问请致电4006-900-900。礼品卡支付金额无法兑换成现金，不能办理现金退款。
