退货时限 7 天　 　 　 换货时限 15 天

1、退换货时限是从商品签收之日起开始计算，尚品网会根据您的物流单号帮您确认您的签收时间。

2、手表类商品均属于高档特殊商品， 购买成功后尚品网会填写表款专属的保修卡，并注明购买日期（以尚品网库房实际发货日期为准）。凡保修卡填写完整售出后，都无法进行涂改，一经涂改便影响该商品的正常保修及鉴定。所以，此种情况下，非质量问题，不给予退换。

3、手表类商品出现外观问题（不属于国家三包政策规定的质量问题），如：如壳、面、针、柄头、玻璃，外观固定件及表带等，不予办理退货。自签收日期起7日内，出现上述情况时，尚品网可提供对应的维修服务，同类问题二次以上修理仍不能正常使用，可免费更换该部件。

4、手表类商品自签收日期7日内，出现国家三包政策规定的质量问题，经鉴定属实，尚品网可以为您提供退换货或维修服务。（遇到国家法定节假日将顺延退换货时间）。

5、手表类商品不可办理退换货条件：

（1）手表表带经过调整或佩戴使用过。
（2）非产品本身质量问题不能办理退换货。
（3）商品及附件（表盒、保卡等）不完整或有损毁。
（4）商品的外包装、说明书、保修单、发票任一缺失。
（5）任何因非正常使用及保管出现质量问题的商品（碰撞、划伤、自行拆卸、维修及不正确的操作）。
（6）手表类商品出现外观问题（不属于国家三包政策规定的质量问题），如：如壳、面、针、柄头、玻璃，外观固定件及表带等。
特别提示：手表类商品，无质量问题不提供退换货服务，给您造成的不便望您谅解。
