退货时限 7 天

1、退换货时限是从商品签收之日起开始计算，尚品网会根据您的物流单号帮您确认您的签收时间。

2、服装鞋靴类商品自签收之日起7日内，如遇商品质量问题（除人为、客观外力因素）,可申请退换货。在退换货时，请保证商品包装吊牌完整、无污渍破损、未经使用、不影响二次销售并提供商品购买凭证，方可申请办理退换货。

3、退换货商品指导。

面料问题：
开裂，面料不牢，拔丝。有洞（做旧设计除外），破裂。
工艺问题：
脱线，缺口，印花不良，裤，袖有短有长，拉链，钮扣及其他配件损坏。
水洗问题：
严重褪色变色（部分商品天然落色，如牛仔），变形。
其他问题：
鞋类的断底，断面，掉跟，掉漆，开胶，开线等。
特别提示：内衣类商品不提供试穿服务，商品一经签收无质量问题不提供退换货服务，给您造成的不便望您谅解。
